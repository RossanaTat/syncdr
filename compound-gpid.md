# syncdr

## Objective

syncdr is an R package for comparing and synchronizing directories by detecting differences in file content or metadata and safely propagating updates (including copying, overwriting, and deleting files). It is designed for R users—especially data analysts and teams—who manage complex file-based workflows across local and shared environments and need reproducible, transparent file management.

## Key Deliverables

- R package (syncdr)
- Directory comparison outputs (tables/tibbles summarizing file differences)
- Duplicate detection reports (lists/tables of identical files by content)
- Synchronization action plans (preview tables of files to copy/delete/overwrite)
- Console outputs (structured summaries, progress messages, directory trees)
- Optional backup file sets (copies of files before overwrite/delete, if enabled)
